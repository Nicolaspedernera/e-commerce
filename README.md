# E-commerce API

1. Clone the repository.
2. Run:
```
npm install
```
3. Have Nest CLI installed.
```
npm i -g @nestjs/cli
```
4. Clone the file ```.env.template``` and rename to ```.env```
5. Fill environment variables.
6. Run the data base.
```
docker-compose up -d
```
7. Run Dev : ```npm run start:dev```
